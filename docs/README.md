# ClawFi Documentation

Welcome to the official ClawFi documentation. ClawFi is a comprehensive crypto intelligence platform that helps traders identify risks, track signals, and make informed decisions across multiple blockchains.

## What is ClawFi?

ClawFi provides real-time crypto intelligence through:

- **Browser Extension** - Injects risk analysis directly into popular DeFi platforms
- **SDK & API** - Programmatic access to token analysis and signals
- **Signal Detection** - Real-time alerts for whale movements, rug pulls, and more
- **Multi-Chain Support** - Works across Ethereum, BSC, Solana, Base, and 20+ chains
